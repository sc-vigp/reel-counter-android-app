# Screenshots Directory

This directory contains screenshots documenting the development journey across all PRs.

## Structure

```
.screenshots/
├── pr1/              # PR #1 screenshots
├── pr2/              # PR #2 screenshots
├── pr3/              # PR #3 screenshots
└── ...
```

## How to Add Screenshots

1. Transfer screenshots from your phone to PC
2. Save in the appropriate `prX` folder
3. Use descriptive naming: `01-feature-name.png`
4. Update `DEVELOPMENT_JOURNEY.md` with descriptions

## Recommended Format

- **Format:** PNG or JPG
- **Resolution:** Original phone resolution (no need to resize)
- **Size:** Reasonable (< 2MB per screenshot if possible)
- **Naming:** `##-descriptive-name.png` (e.g., `01-app-launch.png`)
